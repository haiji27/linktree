Index.html
html lang="en" digunakan untuk menunjukkan bahwa bahasa utama pada halaman web ini adalah Bahasa Inggris. Bagian head berfungsi untuk menyimpan informasi dan pengaturan halaman web yang tidak ditampilkan langsung ke layar. Meta charset="UTF-8" berfungsi agar karakter, simbol, dan emoji dapat ditampilkan dengan benar oleh browser. Meta name="viewport" digunakan agar tampilan website menjadi responsif dan menyesuaikan ukuran layar pada berbagai perangkat seperti handphone dan laptop. Title My profil berfungsi untuk menentukan judul halaman yang muncul pada tab browser. Link stylesheet digunakan untuk menghubungkan file CSS eksternal yang berfungsi mempercantik tampilan website. Selanjutnya, link Google Fonts digunakan untuk menambahkan jenis font khusus agar tampilan teks lebih menarik, dan link Font Awesome digunakan untuk menampilkan ikon media sosial seperti Instagram, Facebook, Twitter, dan GitHub. Bagian body berisi seluruh konten utama yang ditampilkan pada halaman web. Div dengan class container digunakan sebagai pembungkus utama seluruh elemen agar lebih mudah diatur menggunakan CSS. Pada bagian profil, digunakan tag img untuk menampilkan foto profil dari file lokal, dimana src menunjukkan nama file gambar dan alt berfungsi sebagai teks alternatif jika gambar tidak dapat ditampilkan. Tag h1 digunakan sebagai judul utama yang berisi perkenalan singkat pemilik website, sedangkan class fontsaya digunakan untuk menerapkan gaya font dari CSS. Selanjutnya terdapat bagian social yang berfungsi untuk menampilkan daftar media sosial. Tag h1 digunakan sebagai judul bagian media sosial. Tag a digunakan untuk membuat link yang mengarah ke akun media sosial seperti Instagram, Facebook, Twitter, dan GitHub. Di dalam setiap link terdapat div dengan class sosmed yang berfungsi sebagai tombol media sosial, serta tag i yang digunakan untuk menampilkan ikon media sosial dari Font Awesome. Setiap item media sosial dibuat agar dapat diklik dan langsung mengarahkan pengguna ke akun yang dituju. Di akhir dokumen, penutup body menandakan akhir dari seluruh konten halaman web, sedangkan penutup html menandakan bahwa struktur halaman web telah selesai sepenuhnya dan siap ditampilkan oleh browser.
