**Index.html**  
html lang="en" digunakan untuk menunjukkan bahwa bahasa utama pada halaman web ini adalah Bahasa Inggris. Bagian head berfungsi untuk menyimpan informasi dan pengaturan halaman web yang tidak ditampilkan langsung ke layar. Meta charset="UTF-8" berfungsi agar karakter, simbol, dan emoji dapat ditampilkan dengan benar oleh browser. Meta name="viewport" digunakan agar tampilan website menjadi responsif dan menyesuaikan ukuran layar pada berbagai perangkat seperti handphone dan laptop. Title My profil berfungsi untuk menentukan judul halaman yang muncul pada tab browser. Link stylesheet digunakan untuk menghubungkan file CSS eksternal yang berfungsi mempercantik tampilan website. Selanjutnya, link Google Fonts digunakan untuk menambahkan jenis font khusus agar tampilan teks lebih menarik, dan link Font Awesome digunakan untuk menampilkan ikon media sosial seperti Instagram, Facebook, Twitter, dan GitHub. Bagian body berisi seluruh konten utama yang ditampilkan pada halaman web. Div dengan class container digunakan sebagai pembungkus utama seluruh elemen agar lebih mudah diatur menggunakan CSS. Pada bagian profil, digunakan tag img untuk menampilkan foto profil dari file lokal, dimana src menunjukkan nama file gambar dan alt berfungsi sebagai teks alternatif jika gambar tidak dapat ditampilkan. Tag h1 digunakan sebagai judul utama yang berisi perkenalan singkat pemilik website, sedangkan class fontsaya digunakan untuk menerapkan gaya font dari CSS. Selanjutnya terdapat bagian social yang berfungsi untuk menampilkan daftar media sosial. Tag h1 digunakan sebagai judul bagian media sosial. Tag a digunakan untuk membuat link yang mengarah ke akun media sosial seperti Instagram, Facebook, Twitter, dan GitHub. Di dalam setiap link terdapat div dengan class sosmed yang berfungsi sebagai tombol media sosial, serta tag i yang digunakan untuk menampilkan ikon media sosial dari Font Awesome. Setiap item media sosial dibuat agar dapat diklik dan langsung mengarahkan pengguna ke akun yang dituju. Di akhir dokumen, penutup body menandakan akhir dari seluruh konten halaman web, sedangkan penutup html menandakan bahwa struktur halaman web telah selesai sepenuhnya dan siap ditampilkan oleh browser.  
**style.css**  
Kode style.css digunakan untuk mengatur tampilan dan tata letak halaman web agar terlihat rapi dan menarik. Pada bagian awal, pengaturan margin dan padding diatur menjadi nol untuk menghilangkan jarak bawaan dari browser sehingga tampilan lebih konsisten. Selanjutnya, bagian body diatur dengan tinggi satu layar penuh (100vh) dan lebar penuh, serta menggunakan background image dari file lokal yang diposisikan di tengah, diperbesar menyesuaikan layar, dan tidak diulang. Body juga menggunakan flexbox untuk memposisikan seluruh konten agar berada tepat di tengah halaman secara horizontal dan vertikal. Class container berfungsi sebagai pembungkus utama yang mengatur elemen di dalamnya secara vertikal menggunakan flex-direction column, dengan jarak antar elemen yang diatur menggunakan gap. Class profil digunakan untuk mengatur tampilan bagian profil agar elemen gambar dan teks tersusun ke bawah dan berada di tengah. Pada bagian profil img, ukuran gambar diatur menjadi 150x150 piksel dan diberi border-radius 50% sehingga gambar berbentuk lingkaran seperti foto profil. Pada profil h1, warna teks diatur menjadi pink, diberi latar belakang hitam, padding, dan border-radius agar terlihat seperti label, serta ukuran font diperbesar agar lebih menonjol. Class social digunakan untuk mengatur bagian media sosial agar tersusun vertikal, rata tengah, dan memiliki jarak antar item. Ukuran font diperkecil agar tampilan lebih proporsional. Pada social h1, warna teks diatur menjadi aliceblue dan ukuran font diperbesar sebagai judul bagian. Social a digunakan untuk menghilangkan garis bawah pada link dan mengatur warna teks agar tetap rapi. Class sosmed berfungsi sebagai tombol media sosial, dimana latar belakang diatur berwarna pink, ukuran tombol ditentukan, dan konten di dalamnya diposisikan ke tengah menggunakan flexbox. Properti position relative digunakan agar ikon dapat diposisikan secara bebas di dalam tombol. Pada sosmed i, ikon diposisikan di sebelah kiri dan diperbesar agar mudah dikenali. Efek hover pada sosmed digunakan untuk memberikan perubahan warna saat kursor diarahkan ke tombol, sehingga memberikan efek interaktif dengan transisi yang halus. Class fontsaya digunakan untuk mengatur jenis font menggunakan Google Fonts “Playwrite NZ Basic” agar tampilan teks lebih unik dan konsisten di seluruh halaman, serta mengatur gaya dan ukuran font sesuai desain yang diinginkan.
